Version 1.0

- 기지 순서 변경 헬퍼 기능이 오픈되었습니다.

- 함선 순서 변경 헬퍼 기능이 오픈되었습니다.

Version 1.1

- 멀티툴 순서 변경 헬퍼 기능이 오픈되었습니다.

- 반려동물 순서 변경 헬퍼 기능이 오픈되었습니다.

Version 1.2

- 게임상에서 소환(선택)된 함선의 순서를 변경하더라도 소환 상태를 유지하도록 수정되었습니다.

- 멀티툴 순서를 변경하여도 선택 상태를 유지하도록 수정되었습니다.

- 멀티툴 순서를 변경 시 (Empty slot)이 멀티툴 앞이나 사이에 위치하면 문제가 발생하므로 불가능하게 수정되었습니다.

Version 1.3

- 반려동물 순서 변경 시 산란(알), 커스터마이징 정보도 같이 변경되도록 수정되었습니다.

Version 1.4

- 기지순서 및 함선 순서 변경 시 코르벳의 순서 변경이 가능하도록 수정하였습니다.

- 코르벳 외형 공유 및 전체 공유목록 조회, 공유된 코르벳 외형 적용 기능이 추가되었습니다.

Version 1.5

- 코르벳 외형 적용 시 함선 index에 맞게 데이터를 제대로 생성하지 못하여 게임상에서 코르벳 외형이 보이지 않는 문제를 수정했습니다.

- UI의 사소한 오류와 텍스트를 수정했습니다.
