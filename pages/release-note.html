Version 1.0

- 기지 순서 변경 헬퍼 기능이 오픈되었습니다.

- 함선 순서 변경 헬퍼 기능이 오픈되었습니다.

Version 1.1

- 멀티툴 순서 변경 헬퍼 기능이 오픈되었습니다.

- 반려동물 순서 변경 헬퍼 기능이 오픈되었습니다.

Version 1.2

- 게임상에서 소환(선택)된 함선의 순서를 변경하더라도 소환 상태를 유지하도록 수정되었습니다.

- 멀티툴 순서를 변경하여도 선택 상태를 유지하도록 수정되었습니다.

- 멀티툴 순서를 변경 시 (Empty slot)이 멀티툴 앞이나 사이에 위치하면 문제가 발생하므로 불가능하게 수정되었습니다.

Version 1.3

- 반려동물 순서 변경 시 산란(알), 커스터마이징 정보도 같이 변경되도록 수정되었습니다.

Version 1.4

- 기지순서 및 함선 순서 변경 시 코르벳의 순서 변경이 가능하도록 수정하였습니다.

- 코르벳 디자인 공유 및 전체 공유목록 조회, 공유된 코르벳 디자인 적용 기능이 추가되었습니다.

Version 1.5

- 코르벳 디자인 적용 시 함선 index에 맞게 데이터를 제대로 생성하지 못하여 게임상에서 코르벳이 보이지 않는 문제를 수정했습니다.

- UI의 사소한 오류와 텍스트를 수정했습니다.

Version 1.6

- 코르벳 디자인 공유 업로드 시 게임 스크린샷 여부를 최소한만 검증하도록 추가하였습니다.

- UI의 사소한 오류와 텍스트를 수정했습니다.

Version 1.7

- '[goatfungus] NMSSaveEditor' 사용 시 발생할 수 있는 'SyntaxError: Bad escaped character in JSON' 문제를 수정하였습니다.

* 이 문제를 해결하기 위해 Javascript 기본 JSON 객체대신 커스텀 JSON 객체를 구현했습니다.

* 문제 해결을 위해 데이터 제공 및 피드백을 주신 유튜브 유저 'GreaseMonkey663'님께 감사드립니다.

- 순서변경 기능 페이지에서 JSON 붙여넣기 시 발생하던 오류를 수정하였습니다.

- UI의 사소한 오류와 텍스트를 수정했습니다.

Version 1.8

- 디스플레이 해상도가 낮거나 배율이 높은 경우 화면의 레이아웃에 문제가 생겨 코르벳 목록 보기 등의 기능을 이용할 수 없는 문제를 수정했습니다.

* 문제 해결을 위해 디스플레이 정보 및 피드백을 주신 유튜브 유저 'GreaseMonkey663'님께 감사드립니다.
