노맨즈스카이의 여행자분들께...

노맨즈스카이 플레이 타임도 1000시간이 넘었네요. 너무 재밌어서 하나 더 사서 계정도 두개로 하고 있고...

짧다면 짧고 길다면 긴 플레이타임이지만 하다보니 이런저런 꼼수만 늘었습니다.

그래도 노맨즈스카이 여행자분들께 도움이 되는 일을 하게 되어 기쁩니다.


기지 순서 및 함선 순서를 변경하고 싶어하시는 분들을 많이 봐왔습니다.

에디터를 사용하면 되긴 하는데 함선의 경우 철제독수리나 커스텀 함선은 그냥 데이터 수정하면 외형이 지맘대로 바뀌어 버립니다.

외형을 원래대로 유지하면서 순서를 변경하는 방법은 데이터 분석해서 알고는 있었는데 설명이 힘들어서 그냥 헬퍼 사이트를 만들게 되었습니다.

사이트를 3일정도 틈틈이 만들어서 버그가 있을지도 모릅니다. 반드시 가이드를 읽어주시고 백업한후에 진행하시길 바랍니다.


그리고 노맨즈스카이 때문에 유튜브도 시작했는데 영상 제작은 처음이라 하나 만드는데도 오래걸리고 힘드네요.

시간이 허락된다면 노맨즈스카이 추천 모드도 영상 제작해 보려고 합니다.

제가 모드를 30개 넘게 깔고 플레이 하고 있더라고요...

본업이 있어 제작이 더 느리겠지만 아무튼 많은 응원 부탁드립니다.


감사합니다. 즐거운 여행 되시길!

세이브 데이터 백업

C:\Users\(윈도우 로그인 아이디)\AppData\Roaming\HelloGames\NMS 디렉토리에 있는 노맨즈스카이 Steam 세이브 디렉토리를 통째로 압축이나 복사해서 백업하시면 됩니다.

[goatfungus] NMSSaveEditor

개인적인 느낌은 JSON 에디터시에 속도가 빠르고 깔끔함이 좋았던 에디터입니다. 아래 경로에서 다운로드 받으시면 됩니다.

https://github.com/goatfungus/NMSSaveEditor

설치후 실행하시면 아래와 같은 화면이 나옵니다.

Game Slot에서 에디트할 세이브를 선택해 줍니다.

'save.hg'가 자동저장이고 save2.hg가 수동저장(복원지점)입니다.

게임을 실행하면 둘중에 마지막에 저장된 파일을 로드하므로 엉뚱한 파일을 수정한 후 적용되지 않는다고 생각하시면 안됩니다.

참고로 세이브가 여러개일 경우 두번째 세이브에는 'save3.hg'가 자동저장 'save4.hg가 수동저장입니다. 이런식으로 계속 증가합니다.

Edit Raw JSON을 실행합니다.

열린 JSON Editor 창에서 BaseContext를 클릭합니다.

우측의 텍스트(JSON 데이터)를 전체선택(ctrl + a)하고 복사(ctrl + c)합니다.

기지 순서/함선 순서 변경 헬퍼를 통해 수정한 데이터를 다시 전체선택하고 붙여넣기(ctrl + v)하여 적용합니다.

창을 닫으려고 하면 데이터가 변경되었다고 저장할거나고 묻는데 '예'를 클릭해 주세요.

에디터를 닫으려고 하면 저장하고 닫을거냐고 묻는데 '예'를 클릭해 주세요.

게임을 실행하고 즐겨 주세요.

[NomNom] A No Man's Sky Multi-Tool

개인적인 느낌은 'goatfungus'보다 속도는 느리지만 한글 지원도 되고 기능의 편의성이 더 좋았던 에디터입니다. 더운로드시 Nexus 모드 사이트에 회원가입이 필요합니다. 아래 경로에서 다운로드 받으시면 됩니다.

https://www.nexusmods.com/nomanssky/mods/1566

설치후 실행하시고 설정을 해줘야 JSON 에디트가 가능합니다.

고급 모드를 활성하 해주세요.

JSON 수정을 클릭하여 실행합니다.

열린 JSON Editor 창에서 BaseContext를 클릭합니다.

우측의 텍스트(JSON 데이터)를 전체선택(ctrl + a)하고 복사(ctrl + c)합니다.

기지 순서/함선 순서 변경 헬퍼를 통해 수정한 데이터를 다시 전체선택하고 붙여넣기(ctrl + v)하여 적용합니다.

창을 닫으려고 하면 데이터가 변경되었다고 적용할거나고 묻는데 '예'를 클릭해 주세요.

변경 사항이 최종 저장되지 않으면 체크박스가 해제되어 있으니 잘 확인하셔야 합니다.

메뉴에서 저장을 해주세요.

이제 최종 저장되어 체크박스가 다시 활성화 되었습니다.

게임을 실행하고 즐겨 주세요.
