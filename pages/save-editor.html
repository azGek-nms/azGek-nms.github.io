세이브 데이터 백업

C:\Users\(윈도우 계정)\AppData\Roaming\HelloGames\NMS 디렉토리에 있는 노맨즈스카이 Steam 세이브 디렉토리를 통째로 압축이나 복사해서 백업하시면 됩니다.

[goatfungus] NMSSaveEditor

개인적인 느낌은 JSON 에디터 시에 속도가 빠르고 깔끔함이 좋았던 에디터입니다. 아래 URL에서 다운로드하시면 됩니다.

https://github.com/goatfungus/NMSSaveEditor

설치 후 실행하시면 위와 같은 화면이 나옵니다.

Game Slot에서 에디트할 세이브를 선택해 줍니다.

'save.hg'가 자동 저장이고 save2.hg가 수동 저장(복원지점)입니다.

게임을 실행하면 둘 중에 마지막에 저장된 파일을 로드하므로 엉뚱한 파일을 수정한 후 적용되지 않는다고 생각하시면 안 됩니다.

참고로 세이브가 여러 개일 경우 두 번째 세이브에는 'save3.hg'가 자동 저장 'save4.hg가 수동 저장입니다. 이런 식으로 계속 증가합니다.

Edit Raw JSON을 실행합니다.

열린 JSON Editor 창에서 BaseContext를 클릭합니다.

우측의 텍스트(JSON 데이터)를 전체 선택(ctrl + a)하고 복사(ctrl + c)합니다.

기지 순서/함선 순서 변경 헬퍼를 통해 수정한 데이터를 다시 전체 선택하고 붙여넣기(ctrl + v)하여 적용합니다.

창을 닫으려고 하면 데이터가 변경되었다고 저장 여부를 묻는데 '예'를 클릭해 주세요.

에디터를 닫으려고 하면 저장 여부를 묻는데 '예'를 클릭해 주세요.

게임을 실행하고 즐겨 주세요.

[NomNom] A No Man's Sky Multi-Tool

개인적인 느낌은 'goatfungus'보다 속도는 느리지만 한글 지원도 되고 기능의 편의성이 더 좋았던 에디터입니다. 다운로드시 Nexus 모드 사이트에 회원가입이 필요합니다. 아래 URL에서 다운로드하시면 됩니다.

https://www.nexusmods.com/nomanssky/mods/1566

설치 후 실행하시고 설정을 해줘야 JSON 에디트가 가능합니다.

고급 모드를 활성화해주세요.

JSON 수정을 클릭하여 실행합니다.

열린 JSON Editor 창에서 BaseContext를 클릭합니다.

우측의 텍스트(JSON 데이터)를 전체 선택(ctrl + a)하고 복사(ctrl + c)합니다.

기지 순서/함선 순서 변경 헬퍼를 통해 수정한 데이터를 다시 전체 선택하고 붙여넣기(ctrl + v)하여 적용합니다.

창을 닫으려고 하면 데이터가 변경되었다고 적용 여부를 묻는데 '예'를 클릭해 주세요.

변경 사항이 최종 저장되지 않으면 체크박스가 해제되어 있으니 잘 확인하셔야 합니다.

메뉴에서 저장을 해주세요.

이제 최종 저장되어 체크박스가 다시 활성화되었습니다.

게임을 실행하고 즐겨 주세요.
